Configure via DWS

OAuth3 clients and credentials are managed through the DWS control plane. Register your application, configure callback URLs, and manage access tokens all in one place.

1 Open DWS Dashboard
2 Navigate to OAuth3 section
3 Register your application
Open DWS Configuration 
// Redirect to OAuth3
const authUrl = new URL('https://auth.jejunetwork.org/oauth/authorize')
authUrl.searchParams.set('client_id', YOUR_CLIENT_ID)
authUrl.searchParams.set('redirect_uri', YOUR_CALLBACK_URL)
authUrl.searchParams.set('response_type', 'code')

window.location.href = authUrl.toString()

// Handle callback
const code = new URLSearchParams(location.search).get('code')
const tokens = await fetch('/oauth/token', {
  method: 'POST',
  body: JSON.stringify({
    grant_type: 'authorization_code',
    code,
    client_id: YOUR_CLIENT_ID
  })
}).then(r => r.json())
