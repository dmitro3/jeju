# Applications

Jeju includes these applications:

## User-Facing

| App | Port | Description |
|-----|------|-------------|
| [Gateway](/applications/gateway) | 4001 | Bridge, staking, token registry, JNS |
| [Bazaar](/applications/bazaar) | 4006 | DEX, NFT marketplace, launchpad |

## Developer Tools

| App | Port | Description |
|-----|------|-------------|
| [Factory](/applications/factory) | 4008 | Contract templates, deployment |
| [Indexer](/applications/indexer) | 4350 | GraphQL API |

## Infrastructure

| App | Port | Description |
|-----|------|-------------|
| [DWS](/applications/dws) | 4007 | Compute, Storage, CDN |
| [Crucible](/applications/crucible) | 4020 | Agent orchestration |
| [Monitoring](/applications/monitoring) | 4040 | Prometheus, Grafana |

## Running Apps

Start all apps:

```bash
bun run dev
```

Start individually:

```bash
cd apps/gateway
bun run dev
```

## App Manifest

Each app has a `jeju-manifest.json`:

```json
{
  "name": "gateway",
  "type": "core",
  "commands": { "dev": "bun run dev" },
  "ports": { "main": 4001 },
  "autoStart": true
}
```
